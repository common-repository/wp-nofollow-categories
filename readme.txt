Nofollows category links across the site. Adds Noindex to category pages.


== Description == 

WP Nofollow Categories is a SEO plugin which adds nofollow attribute to all category links throughout the blog. It will also add noindex tag to category page and nofollow author links. This will effectively remove the category pages from SEO point of view. 

The goal is to flow the page rank to other pages on your site like posts and tags while avoiding duplicate content issue.

Plugin by <a href="http://www.prelovac.com/vladimir">Vladimir Prelovac</a>. 
== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Go to the Plugins page and activate the plugin.
3. That's all, everything will work automatically
